# Program 8a - Cookie Servlet

## Objective
Create a servlet program to store and retrieve user name using cookies.

## Files
- CookieServlet.java
- index.html

## Source Code
- Servlet: https://github.com/musabnova/Program8a/blob/master/src/main/java/com/cookieservlet/CookieServlet.java
- HTML: https://github.com/musabnova/Program8a/blob/master/src/main/webapp/index.html

## How to Run
1. Run project on Apache Tomcat
2. Open: http://localhost:8080/Program8a/
3. Enter your name and submit
4. Cookie will be created and displayed
